# Codagent Ecosystem — Naming

## Ecosystem

- **GitHub org / brand / LLC**: `codagent`
- **Full display name**: Codagent
- **Portmanteau**: cod(e) + agent
- **Double meaning**: code + agent (an agent that codes) + co-agent (a collaborative partner)

## Agvent (This Project)

- **CLI / npm package / repo**: `agvent`
- **Full display name**: Agent Venture
- **Portmanteau**: ag(ent) + (ad)venture
- **Role in ecosystem**: Orchestration engine — the core workflow runner

### Double Meaning

"Venture" works on two registers:

1. **Game/RPG**: A venture is a quest, an expedition, a dangerous journey into the unknown. Workflows become ventures that agents embark on — levels to advance through, challenges to overcome.

2. **Business/Professional**: A venture is an entrepreneurial undertaking. This reflects the project's ambition — a serious tool for building the best agentic coding workflows, not just a toy.

## Gamification Theme

The UI direction draws from classic Final Fantasy games (pixel-era aesthetic). Core concept: workflows are reimagined as game levels that you advance through.

| Agvent Concept | Gamified Concept |
|----------------|-----------------|
| Workflow | Venture / quest |
| Workflow step | Level / stage |
| Step completion | Level cleared |
| Agent session | Party member / hero |
| Orchestrator | Game master |

The Final Fantasy influence extends to UX and visual theme — retro pixel art aesthetic, RPG-style progression, and terminology drawn from classic FF game systems.
